Privacy Policy

Last updated: June 10, 2026

1. Overview

pipeline-dev ("the Service") is a personal content automation tool. This policy describes how data is handled when the Service interacts with the TikTok API.

2. Data We Collect

The Service collects and stores only the following:

OAuth Access Token: A TikTok API access token used to authenticate video uploads. This token is stored locally on the developer's machine in an environment file (.env) and is never transmitted to any third party.

The Service does not collect any personal information from TikTok users, viewers, or any third parties.

3. Data We Do Not Collect

The Service does not collect, access, or store: usernames, email addresses, profile information, viewing history, analytics data, follower lists, direct messages, or any other TikTok user data.

4. How Data Is Used

The OAuth token is used solely to authenticate API requests to upload and publish video content to the developer's own TikTok account via the Content Posting API.

5. Data Sharing

No data is shared with third parties. The Service operates entirely on the developer's local machine and communicates only with the TikTok API.

6. Data Retention

The OAuth token is retained locally until manually revoked or the application is deleted. No other data is retained.

7. Security

The OAuth token is stored in a local environment file excluded from version control (.gitignore). Access is limited to the developer's machine.

8. Children's Privacy

The Service is not directed at children under 13 and does not knowingly collect data from minors.

9. Changes

This policy may be updated at any time. Changes take effect immediately upon posting.
